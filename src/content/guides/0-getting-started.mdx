# Getting Started with StudyDocs

Welcome to StudyDocs! This guide will help you get started with creating and organizing your study materials.

## What is StudyDocs?

StudyDocs is a modern documentation platform designed specifically for students and educators. It provides an intuitive way to create, organize, and share study materials using Markdown and MDX.

## Key Features

- **Modern Design**: Clean, responsive interface optimized for reading and learning
- **Markdown & MDX Support**: Write content in Markdown with extended features
- **Code Highlighting**: Beautiful syntax highlighting for code snippets
- **Math Equations**: LaTeX-style math equations using KaTeX
- **Dark Mode**: Comfortable reading in any lighting condition

## Quick Start

1. Create a new document:

```bash
npm run new-doc
```

2. Write your content in MDX:

```mdx
---
title: My Study Notes
description: Notes from today's lecture
author: Your Name
---

# Introduction to React Hooks

React Hooks are functions that let you use state and other React features
in functional components.
```

3. Add math equations using LaTeX syntax:

```latex
When \(a \ne 0\), there are two solutions to \(ax^2 + bx + c = 0\):

\[x = {-b \pm \sqrt{b^2-4ac} \over 2a}\]
```

## Organization Tips

1. **Use Categories**: Organize content into logical categories
2. **Add Tags**: Tag content for better searchability
3. **Link Related Content**: Create connections between related documents
4. **Regular Updates**: Keep your study materials up to date
